## UIインタラクション

### キーバインド

- `Tab` / `Shift+Tab`: フィールド間の移動
- `↑` / `↓`: ドロップダウンやラジオボタンの選択
- `Space`: チェックボックスの切り替え
- `Enter`: 決定/実行
- `Esc`: 前の画面に戻る
- `q` / `Ctrl+C`: アプリケーション終了

### フォームバリデーション

- エンドポイントが空の場合はエラー表示
- ポート番号が1-65535の範囲外の場合はエラー表示
- Compartment IDの形式チェック（Cloud版）
- 入力エラーがある場合は「接続する」ボタンを無効化

### カラーテーマ（予定）

- アクセントカラー: シアン (#00D9FF)
- 成功: グリーン (#00FF00)
- エラー: レッド (#FF0000)
- 警告: イエロー (#FFFF00)
- テキスト: ホワイト/グレー
- 選択中のフィールド: ハイライト表示
