### 接続状態の表示パターン（実装済み）

#### 接続テスト中
ステータス領域に "Connecting..." と緑色で表示

#### 接続成功時
ステータス領域に "Connected" と緑色で表示

#### 接続失敗時
ステータス領域に "Connection failed" または詳細なエラーメッセージを赤色で表示（画面幅に応じて切り詰め）

## UIインタラクション（実装済み）

### キーバインド

**エディション選択画面:**
- `Tab` / `Shift+Tab` または `↑` / `↓`: 選択肢の移動（ループ）
- `Enter`: 決定
- `q` / `Ctrl+C`: アプリケーション終了

**On-Premise接続設定画面:**
- `Tab` / `Shift+Tab`: フィールド間の移動（ループ）
- `Space`: チェックボックスの切り替え
- `Backspace`: テキスト入力の削除
- 文字キー: テキスト入力
- `Enter`: ボタン実行（接続テスト/接続する）
- `Esc`: エディション選択画面に戻る
- `Ctrl+C`: アプリケーション終了

### カラーテーマ（実装済み）

- **アクセントカラー**: シアン (#00D9FF) - ボーダー、選択中の要素
- **成功**: グリーン (#00FF00) - 接続成功メッセージ
- **エラー**: レッド (#FF0000) - 接続失敗メッセージ
- **背景**: ダークグレー (#1A1A2E) - ヘッダー、ステータスバー
- **テキスト**: ホワイト (#FFFFFF) - サブタイトル、通常のテキスト
- **グレー**: (#888888) - ヘッダー、ラベル、ヘルプテキスト
